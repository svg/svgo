Removes hrefs to JavaScript URIs, including unconventional namespaces, even when
the JavaScript URI is written with uppercase characters.

===

<svg xmlns="http://www.w3.org/2000/svg" xmlns:uwu="http://www.w3.org/1999/xlink" viewBox="0 0 100 100" version="1.1">
  <a href="JAVASCRIPT:(() => { alert('Default namespace with uppercase URI scheme.') })();">
    <text y="20">uwu</text>
  </a>
  <a uwu:href="JavaScript:(() => { alert('Explicit/prefixed namespace with mixed-case URI scheme.') })();">
    <text y="30">uwu</text>
  </a>
</svg>
