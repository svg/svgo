---
title: Apply Transforms to Shapes
svgo:
  pluginId: applyTransformsShapes
  parameters:
    floatPrecision:
      description: Number of decimal places to round to, using conventional rounding rules.
      default: 3
    leadingZero:
      description: If to trim leading zeros.
      default: true
  defaultPlugin: true
---

Applies the `transform` attribute directly to `<circle>`, `<ellipse>`, `<rect>`, and `<image>` when possible.
